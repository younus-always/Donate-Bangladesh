What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web document. DOM is a tree-like structure of nodes and objects.DOM can do anything in an HTML Document like Access, Change, Delete or Add etc.

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation is a pattern based concept of Event Bubbling. It's an event-handling pattern it's allows you to handle events at a higher level in the DOM tree. It's can efficient and can simplify event-handling.

How do you manipulate an element's attributes and styles using the DOM?

Manipulate an element's attributes and styles you can use methods like setAttribute() to set attributes (disabled, autoplay) getElementById() to target an element, add() adding a class or remove() to remove a class of the classList object. So you can manipulate element's by using Javascript.
